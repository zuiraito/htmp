# Installation
## Dependencies
- `gpxbabel`
- `folium` (python lib)
```
git clone https://github.com/zuiraito/htmp.git
cd htmp/
chmod +x run.sh
chmod +x convert.sh
```
Change input file path in `convert.sh`.
# Running
```
run.sh
```
